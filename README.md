## How To Use

```bash
# Clone this repository
$ git clone https://github.com/MEsen1/Django_Rest_ToDo.git

# Install dependencies
> python -m venv env
> env/Scripts/activate
> pip install -r requirements.txt
> python manage.py migrate

# Run the app
> python manage.py runserver
```
